Welcome to The Todo List Simple Server!

Instructions:

Note: Replace {base_url} with your actual server URL (e.g., http://localhost:3000).

1. Add a Task

Perform a POST request to {base_url}/addTask with your task details in the following format:

{
    "description": "Your task description"
}

A unique ID and a default status of WAITING (-1) will be assigned automatically.

2. View All Tasks

Perform a GET request to {base_url}/tasks to see all tasks in the todo-list.

3. View a Specific Task

Perform a GET request to {base_url}/tasks/{id} where {id} is the ID of the task you want to view.

4. Delete a Task

Perform a DELETE request to {base_url}/deleteTask/{id} where {id} is the ID of the task you want to delete.

5. Update a Task Description

Perform a PUT request to {base_url}/putTask/{id} where {id} is the ID of the task you want to update. The request body should include the new description in the following format:

{
    "description": "Updated task description"
}

The status will be reset to WAITING (-1).

6. Update a Task Status

Perform a PATCH request to {base_url}/updateStatus/{id} where {id} is the ID of the task you want to update. The request body should include the new status in the following format:

{
    "status": "new status"
}

Valid status values:

  • -1: WAITING
  • 0: IN PROGRESS
  • 1: COMPLETED
